# My Brew

Bootstrap my Macbook.

[Install homebrew.](http://brew.sh/)

See the [github readme](https://github.com/Homebrew/homebrew-bundle/blob/master/Readme.md)

Install:

```
$ brew bundle
```

## Add Lock to menu bar

```
Spotlight 
  -> Keychain Access 
    -> Preferences 
      -> Show keychain status in menu bar
```

## fn + F*

```
System Preferences
  -> Keyboard
    -> Use all F1, F2, etc
```
